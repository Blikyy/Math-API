# Math-API

 ## f-1 
podporované operace:

 - sum = +
 - sub = -
 - mul = *
 - div = /
 - pow = **

 Příklad odkazu:
 localhost/math-api/f1/index.php/?operation=sum&number=1,2,3,4

výsledek:

{"report":"ok","result":10}
